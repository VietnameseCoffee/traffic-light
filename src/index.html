Traffic

Control a traffic light as cars zoom by on their way to work in New Townsville.

Some cars are faster than others like the sports car zooming by the old Bug

If they're in the same lane they'll have to slow down and follow. Unfortunately New Townsville does not allow lane changes, so cars have to tail slower cars. How this town survives is a mystery

Cars move using several types of JavaScript methods, specifically requestAnimationFrame, setTimeout, setInterval. It showcases the different callback rates when they get called in the event loop, most notably between requestAnimationFrame and setTimeout/setInterval.
